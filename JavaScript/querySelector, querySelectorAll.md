### querySelector, querySelectorAll

*css 선택자를 사용해 엘리먼트 찾기*

```javascript
document.querySelectorAll('div')[0];
document.querySelector('div');

// 둘의 결과는 동일
```

> querySelector( ) -> ElementNode를 반환한다.
>
> querySelectorAll( ) -> NodeList를 반환한다.

#### 브라우저 지원

- Chrome : 4.0+
- Firefox : 3.5+
- Internet Explorer : 8.0+
- Opera : 10.0+
- Safari : 3.2+
