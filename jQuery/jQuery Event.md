### jQuery Event

```javascript
$("p").click(function(){
  $(this).hide();
});
```

```javascript
$("p").dbclick(function(){
  $(this).hide();
});
```

```javascript
$("p").mouseenter(function(){
  alert("You entered p!!");
});
```

```javascript
$("p").mouseleave(function(){
  alert("You now leave p!!");
});
```
